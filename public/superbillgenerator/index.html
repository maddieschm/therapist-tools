Printing Instructions

After generating a superbill, click the "Print Superbill" button. In the browser's print dialog, choose "Save as PDF" as the destination to create a digital file for your client.

Export/Import Instructions

Use "Export" to save a client's superbill data as a JSON file for your records. Use "Import" to load data from a previously saved JSON file, pre-filling the form.
